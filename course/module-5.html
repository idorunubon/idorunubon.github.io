Module 5: วิดีโอ & สรุปเนื้อหาเสริม

บททบทวนฉบับกระชับของแนวคิดสำคัญทั้งคอร์ส — แต่ละหัวข้อมีพื้นที่สำหรับฝังวิดีโอประกอบของคุณเอง (แก้บล็อก video-slot เป็น iframe ของ YouTube ได้เลย)

สรุปที่ 1

Claude Skills — สรุปฉบับทบทวน

🎬 พื้นที่ฝังวิดีโอ: Claude Skillsแทนที่บล็อกนี้ด้วย <iframe> วิดีโอ YouTube ของคุณ
  • Skill = สูตรการทำงานถาวร: เปลี่ยนวิธีทำงานที่เวิร์กให้กลายเป็นเครื่องมือที่ AI หยิบใช้เองได้ทุกครั้งที่งานตรงเงื่อนไข
  • โครง 3 ส่วนจำง่าย: ใช้เมื่อไร → ทำอย่างไรทีละขั้น → ผลลัพธ์หน้าตาแบบไหน
  • เริ่มจากงานที่ทำซ้ำและมีสูตร: รายงาน วิเคราะห์ เอกสารมาตรฐาน — งานที่ "ทำเหมือนเดิมทุกครั้ง" คือ Skill ที่ดีที่สุด
  • ให้ Claude สร้าง Skill ให้: ไม่ต้องเขียนเองตั้งแต่ศูนย์ บอกสเปกแล้วให้ AI ประกอบให้ (Module 3, บท Setup)
สรุปที่ 2

Claude Cowork Beginners Guide — เช็กลิสต์ผู้เริ่มต้น

🎬 พื้นที่ฝังวิดีโอ: Cowork Beginners Guideแทนที่บล็อกนี้ด้วย <iframe> วิดีโอ YouTube ของคุณ

เส้นทางเริ่มต้นที่ถูกต้องใน 7 ขั้น (รายละเอียดเต็มอยู่ Module 1):

  1. ติดตั้งแอป Desktop และล็อกอิน (บทที่ 2)
  2. ตั้งค่า Memory ให้ AI รู้จักธุรกิจเรา (บทที่ 2)
  3. จัดโฟลเดอร์ AI-Workspace ให้เรียบร้อยก่อนเริ่ม (บทที่ 2)
  4. ฝึกสั่งงานด้วยโครงสร้าง บทบาท-บริบท-งาน-รูปแบบ (บทที่ 3)
  5. มอบหมายงานแรกให้ Cowork ที่ได้ "ไฟล์ผลลัพธ์จริง" (บทที่ 4)
  6. สั่งเป็นชิ้นงาน + ระบุ Definition of Done เสมอ (บทที่ 4)
  7. สัปดาห์แรกตรวจงาน AI ทุกชิ้น แล้วค่อย ๆ ปล่อยมือ
กับดักมือใหม่ 3 อย่าง: (1) ใช้ Cowork เป็นแค่แชตถาม-ตอบ (2) สั่งงานกำกวมแล้วโทษ AI (3) ไม่ตรวจงานเลยตั้งแต่วันแรก — เลี่ยง 3 อย่างนี้ได้ คุณนำหน้าคนใช้ AI ส่วนใหญ่แล้ว
สรุปที่ 3

5 Use Cases ใน Cowork ที่ใช้ได้ทุกธุรกิจ

🎬 พื้นที่ฝังวิดีโอ: 5 Use Casesแทนที่บล็อกนี้ด้วย <iframe> วิดีโอ YouTube ของคุณ
  1. รวม + วิเคราะห์ไฟล์กระจัดกระจาย: ยอดขายหลายไฟล์ → ไฟล์เดียว + Insight (Module 1 บท 5-6)
  2. รายงานอัตโนมัติรายสัปดาห์: Scheduled Task สร้างรายงานเองทุกเช้าวันจันทร์ (Module 1 บท 9)
  3. Dashboard ผู้บริหาร: เปิดไฟล์เดียวเห็นภาพธุรกิจทั้งหมด พร้อมจุดที่ต้องดู (Module 3, Skill #4)
  4. เครื่องผลิตงานการตลาด: วิเคราะห์คู่แข่ง → หา ICP → เขียนแอด → สร้างหน้าขาย ครบสายพาน (Module 3)
  5. เลขาประชุมอัตโนมัติ: transcript → สรุป + Action Items + อีเมล follow-up (Module 4 บท 3)
สรุปที่ 4

Cowork เหมือนมีทีมงานเพิ่ม 5 คน

🎬 พื้นที่ฝังวิดีโอ: ทีมงานเพิ่ม 5 คนแทนที่บล็อกนี้ด้วย <iframe> วิดีโอ YouTube ของคุณ

ภาพสุดท้ายที่คอร์สนี้อยากให้คุณเห็น: ระบบที่สร้างครบแล้วเทียบเท่าทีมงาน 5 ตำแหน่ง

ตำแหน่งที่ได้มาจากอะไรในคอร์ส
1. นักวิเคราะห์ข้อมูลวิเคราะห์หลายไฟล์ + รวม Excel (Module 1 บท 5-6)
2. ผู้ช่วยผู้บริหารScheduled Tasks + สรุปประชุม + Briefing รายวัน (Module 1 บท 9, Module 4 บท 3)
3. นักการตลาดAdSpy + ICP Profiler + Landing Page Builder (Module 3)
4. นักกลยุทธ์Competitor Snapshot + CEO Dashboard (Module 3)
5. เจ้าหน้าที่ AutomationConnectors + Zapier + n8n (Module 2, Module 4 บท 1)

หัวใจไม่ใช่ "AI แทนคน" แต่คือ คุณกลายเป็นหัวหน้าทีมที่มอบหมายงานเก่ง — เวลาที่เคยหมดไปกับงาน manual ถูกคืนมาให้ใช้กับการตัดสินใจ ความสัมพันธ์กับลูกค้า และการเติบโตของธุรกิจ
