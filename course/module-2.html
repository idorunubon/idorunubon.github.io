Module 2: Cowork Connectors

พลังที่แท้จริงของ AI Agent เกิดขึ้นเมื่อมันเชื่อมกับเครื่องมือที่เราใช้ทำงานทุกวัน — Module นี้พาเชื่อม Claude เข้ากับแอปต่าง ๆ ผ่าน MCP และ Zapier ทั้งบนเว็บและ Desktop

Lesson 1

MCP คืออะไร — ภาษากลางที่ทำให้ AI คุยกับแอปอื่นได้

เป้าหมายบทนี้: เข้าใจแนวคิด MCP และ Connector ก่อนลงมือเชื่อมจริงในบทถัดไป

MCP (Model Context Protocol) คือมาตรฐานกลางที่ทำให้ AI เชื่อมต่อกับแอปและแหล่งข้อมูลภายนอกได้อย่างปลอดภัย — คิดง่าย ๆ ว่าเป็น "หัวปลั๊ก USB ของโลก AI": แอปไหนทำ MCP server ไว้ Claude ก็เสียบใช้ได้เลย

  • MCP server = ตัวแทนของแอปหนึ่ง ๆ ที่เปิด "เครื่องมือ" ให้ AI เรียกใช้ เช่น MCP ของปฏิทินจะมีเครื่องมือ ดูตารางนัด / สร้างนัด
  • Connector = ชื่อเรียกการเชื่อมต่อเหล่านี้ในหน้า Settings ของ Claude — กดเชื่อม ล็อกอินอนุญาตสิทธิ์ แล้วใช้ได้เลย

เมื่อเชื่อมแล้ว สิ่งที่เปลี่ยนไปคือ Claude ทำงาน "ข้ามแอป" ได้ในคำสั่งเดียว เช่น "ดูอีเมลจากลูกค้า A เมื่อวาน สรุปประเด็น แล้วสร้างนัดประชุมทีมพรุ่งนี้ 10 โมง" — อ่านเมล + สรุป + ลงปฏิทิน จบในคำสั่งเดียว

หลักการเลือกว่าจะเชื่อมอะไร: เริ่มจากแอปที่คุณเปิดบ่อยที่สุด 3 อันดับแรกในแต่ละวัน — นั่นคือจุดที่ Connector คืนเวลาให้คุณมากที่สุด
Lesson 2

MCP servers ของ Zapier — ประตูสู่แอปหลายพันตัว

เป้าหมายบทนี้: ใช้ Zapier MCP เชื่อม Claude กับแอปที่ยังไม่มี Connector ตรง

Zapier คือแพลตฟอร์มเชื่อมแอปกว่า 7,000 ตัว (LINE, Google Sheets, Facebook, Shopify, Notion, HubSpot ฯลฯ) และ Zapier มี MCP server ที่ทำให้ Claude สั่งงานแอปเหล่านั้นผ่าน Zapier ได้ — เท่ากับว่าเชื่อม Zapier ตัวเดียว ได้ประตูไปสู่แอปเกือบทุกตัวที่ธุรกิจใช้

ขั้นตอนการตั้งค่า

  1. สมัคร/ล็อกอิน Zapier แล้วไปที่หน้า Zapier MCP (mcp.zapier.com)
  2. สร้าง MCP endpoint ของเรา แล้วเลือกว่าจะเปิด "เครื่องมือ" อะไรให้ AI ใช้บ้าง เช่น เพิ่มแถวใน Google Sheets, ส่งข้อความแจ้งเตือน, สร้าง Task
  3. นำการเชื่อมต่อที่ได้ไปเพิ่มใน Claude ที่หน้า Settings → Connectors → Add custom connector
  4. ทดสอบด้วยคำสั่งง่าย ๆ ก่อน เช่น "เพิ่มแถวทดสอบใน Google Sheets ชื่อ Test"
ทุกครั้งที่ฉันบอกว่า "บันทึกลูกค้าใหม่" ให้ทำสิ่งนี้: 1. ถามข้อมูล: ชื่อ เบอร์โทร สินค้าที่สนใจ ช่องทางที่รู้จักเรา 2. เพิ่มลงชีต "รายชื่อลูกค้า" ผ่าน Zapier (เครื่องมือ Google Sheets: Create Row) 3. ยืนยันกลับมาว่าบันทึกแถวไหน สำเร็จหรือไม่ ตอนนี้: บันทึกลูกค้าใหม่
ข้อควรระวัง: เปิดเครื่องมือใน Zapier MCP เท่าที่ใช้จริงเท่านั้น และเลี่ยงการเปิดเครื่องมือประเภท "ลบข้อมูล" ให้ AI ใช้ — เพิ่ม/อ่าน ปลอดภัยกว่าแก้/ลบ
Lesson 3

เชื่อม Claude กับแอป — ทั้งบนเว็บและ Desktop

เป้าหมายบทนี้: รู้วิธีเพิ่ม Connector ทั้งสองช่องทาง และเลือกใช้ให้ถูกกับลักษณะงาน

บนเว็บ / แอปมือถือ (claude.ai)

  1. ไปที่ Settings → Connectors
  2. เลือกแอปที่มีให้เชื่อมโดยตรง (เช่น Google Drive, Gmail, ปฏิทิน, Notion, Canva) แล้วกด Connect
  3. ล็อกอินและอนุญาตสิทธิ์ของแอปนั้น — เชื่อมครั้งเดียวใช้ได้ทุกแชต
  4. Connector แบบ custom (เช่นจาก Zapier หรือที่เราสร้างเอง) เพิ่มได้จาก Add custom connector

บนแอป Desktop (สำหรับ Cowork)

  1. Connector ที่เชื่อมบนบัญชีจะใช้ได้บน Desktop ด้วย
  2. ส่วน MCP server แบบติดตั้งในเครื่อง (Local MCP) เพิ่มได้จากไฟล์ตั้งค่าของแอป Desktop หรือเมนู Developer/Extensions — เหมาะกับการเชื่อมระบบภายในบริษัทที่ไม่ได้อยู่บนคลาวด์
  3. จุดแข็งของ Desktop คือใช้ Connector ควบคู่กับการเข้าถึงไฟล์ในเครื่อง เช่น "ดึงข้อมูลจากระบบ → ประมวลผล → บันทึกเป็นไฟล์ Excel ในโฟลเดอร์งาน"

เลือกช่องทางให้ถูกงาน

ลักษณะงานใช้ช่องทางไหน
ถาม-ตอบ / สรุปข้อมูลจากแอปคลาวด์ ระหว่างเดินทางเว็บ / มือถือ + Connector
งานที่ต้องอ่าน-สร้างไฟล์จริงในเครื่องDesktop + Cowork
งานข้ามระบบ: ดึงจากแอป → ทำไฟล์ → ส่งต่อDesktop + Cowork + Connector (ครบสุด)
แบบฝึกหัดท้ายบท
เชื่อม Connector อย่างน้อย 2 ตัวที่ตรงกับงานคุณ แล้วทดสอบคำสั่ง "ข้ามแอป" 1 คำสั่ง เช่น สรุปอีเมลสัปดาห์นี้แล้วบันทึกประเด็นลงไฟล์
Lesson 4

List เครื่องมือต่าง ๆ ที่ AI ทำงานได้ด้วยดี

เป้าหมายบทนี้: มีรายการอ้างอิงว่าควรเชื่อมอะไร เพื่องานแบบไหน
หมวดเครื่องมือตัวอย่างงานที่ AI ทำให้ได้
ไฟล์ & เอกสารGoogle Drive, ไฟล์ในเครื่อง (Cowork)ค้นเอกสาร สรุปไฟล์ รวมข้อมูลหลายไฟล์ สร้างรายงาน
อีเมลGmailสรุปเมลค้าง ร่างเมลตอบ ติดตามเมลสำคัญ
ปฏิทินGoogle Calendarดูตารางว่าง สร้างนัด สรุปตารางสัปดาห์
โน้ต & จัดการงานNotion, Asana, Linear, Jiraสร้าง/อัปเดต Task สรุปสถานะโปรเจกต์ เขียนเอกสารทีม
สื่อสารทีมSlack, Microsoft Teamsสรุปแชตที่พลาด ร่างประกาศ ส่งสรุปรายงานเข้าห้องทีม
ออกแบบ & นำเสนอCanva, Figma, Gammaสร้างงานออกแบบ/สไลด์จากเนื้อหาที่ AI เตรียม
ธุรกิจ & การขายHubSpot, Stripe, Shopify (ผ่าน Zapier)สรุปดีล เช็กยอดขาย ติดตามลูกค้า
อื่น ๆ อีกหลายพันแอปผ่าน Zapier MCPLINE Notify, Facebook Lead, Airtable ฯลฯ

หลักการจัดชุดเครื่องมือ (Tool Stack) ของคุณ

  1. ชุดเริ่มต้น (ทุกคนควรมี): ไฟล์ + อีเมล + ปฏิทิน
  2. ชุดตามสายงาน: การตลาดเพิ่ม Canva/Meta, ฝ่ายขายเพิ่ม CRM, ผู้บริหารเพิ่มเครื่องมือรายงาน
  3. ทบทวนทุกเดือน: Connector ไหนไม่ได้ใช้ ให้ถอดออก — ยิ่งน้อยยิ่งปลอดภัยและ AI เลือกเครื่องมือได้แม่นขึ้น
โปรเจกต์ท้าย Module
ออกแบบ Tool Stack ของคุณเอง: เลือกเครื่องมือ 3-5 ตัว เชื่อมให้ครบ แล้วสร้าง workflow ข้ามแอป 1 เส้น เช่น "ทุกเช้าสรุปเมล + ตารางนัดวันนี้ + งานค้าง เป็นข้อความ Briefing เดียว"
